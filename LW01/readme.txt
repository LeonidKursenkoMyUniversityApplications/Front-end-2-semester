Задание 1
Изограмма - это слово, которое не имеет повторяющихся букв, последовательных или не последовательных. Реализуйте функцию, которая определяет, является ли строка, содержащая только буквы, изограммой. Предположить, что пустая строка является изограммой. Игнорировать регистр букв.
Примеры работы функции
// isIsogram( "Dermatoglyphics" ) == true
// isIsogram( "aba" ) == false
// isIsogram( "moOse" ) == false

Задание 2
Напишите функцию, которая принимает положительное целое число и возвращает меньшее положительное целое число, содержащее одни и те же цифры.
Примеры работы функции
// nextSmaller(21) == 12
// nextSmaller(531) == 513
// nextSmaller(20715) == 20571

// nextSmaller(9) == -1
// nextSmaller(111) == -1
// nextSmaller(135) == -1
